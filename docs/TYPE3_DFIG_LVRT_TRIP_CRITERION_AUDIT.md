## 3. 论文正文原意

第 2.3.2 节先说明风电机组在并网点电压跌落或升高时，应在一定电压幅值和时间范围内保持并网运行。正文明确提到：并网点电压跌至额定电压 20% 时，风机应保持并网持续运行 625 ms，并在电压跌落开始后 2 s 内恢复至额定电压 90%。同一节随后引入风机脱网事件，并根据并网点电压 `Vs` 设置电压穿越保护延时动作时间 `tVRT`。

高电压穿越分支也出现在同一公式中，但本轮只为低电压脱网原型做判据审计；高电压、频率保护、低频低压减载和暂态过电压保护均记录为后续范围。

## 4. 公式 (2-8) 转录

根据 PDF 第 20 页的文本抽取和视觉核对，公式 (2-8) 可读为：

```text
tVRT =
  0,                                                   Vs <= 0.20 or Vs >= 1.30
  ((Vs - 0.2) / (0.9 - 0.2)) * (2.0 - 0.625) + 0.625,  0.20 < Vs < 0.90
  0.5,                                                 1.25 <= Vs <= 1.30
  1.0,                                                 1.20 <= Vs <= 1.25
  10.0,                                                1.10 <= Vs <= 1.20
```

变量与单位：

| 符号 | 含义 | 单位/基准 |
|---|---|---|
| `Vs` | 风机并网点电压 | p.u.，以额定电压为基准 |
| `tVRT` | 风机电压穿越保护延时动作时间 | s |

低电压线性段可化简为：

```text
tVRT = 1.9642857142857144 * (Vs - 0.2) + 0.625
```

但该化简只适用于公式给出的开区间：

```text
0.20 < Vs < 0.90
```

## 5. 图 2-3 与公式的对应关系

图 2-3 的低电压穿越区域显示：

- 电压跌落开始时刻为 `t0`。
- 低电压底部平台约为 `0.20 pu`，持续至 `t0 + 0.625 s`。
- 随后允许电压恢复边界线性上升，并在 `t0 + 2 s` 到达约 `0.90 pu`。
- 曲线下方标为允许脱网区域，曲线及其上方为电压穿越区域。

因此，对低电压内部区间而言，图 2-3 与公式的线性段一致：允许持续时间从 `(0.20 pu, 0.625 s)` 线性增加到 `(0.90 pu, 2.0 s)`。

但是，边界点 `Vs = 0.20 pu` 存在证据张力：

- 正文和图 2-3 表达了 `20%` 电压处应保持并网 `625 ms` 的要求。
- 公式 (2-8) 第一行写作 `Vs <= 0.20` 时 `tVRT = 0`。

由于正文、图和公式在 `Vs = 0.20 pu` 的等号归属上不完全一致，本轮不能把完整低电压判据标为可直接实现的最终规则。

## 6. 低电压程序化规则状态

本轮可确认的低电压内部规则为：

```text
For 0.20 < Vs < 0.90:
    tVRT = ((Vs - 0.2) / (0.9 - 0.2)) * (2.0 - 0.625) + 0.625
```

边界处理审计：

| 边界 | 论文证据 | 本轮结论 |
|---|---|---|
| `Vs < 0.20` | 公式第一行给出 `tVRT = 0` | 可作为立即脱网候选规则 |
| `Vs = 0.20` | 正文/图支持 625 ms；公式第一行支持 0 s | 冲突，需人工确认 |
| `0.20 < Vs < 0.90` | 公式和图一致 | 可确认线性延时 |
| `Vs = 0.90` | 公式低压线性段为开区间 `< 0.90`，正文说 2 s 内恢复至 90% | 可作为退出低电压计时的恢复边界候选 |
| `Vs > 0.90` 且未进入高压分支 | 低压公式不覆盖 | 不进入低电压脱网计时 |

程序化实现状态：

```text
implementation_status: manual_confirmation_required
```

原因：`Vs = 0.20 pu` 的等号归属影响最严苛低压边界下是否立即脱网。没有人工确认前，不应把该边界写入 PSCAD 外部脱网逻辑。

## 7. 低压持续时间的计时含义

论文在公式前说明 `tVRT` 是电压穿越保护的延时动作时间。工程转译为：当风机并网点电压落入低电压区域时，根据当前 `Vs` 计算允许持续时间；若低压状态持续超过允许时间，则触发风机脱网事件。

仍需注意：

- 论文片段没有在公式附近详细说明计时器是按瞬时 `Vs` 连续更新、按最小电压锁存，还是按故障期间最严重电压确定。
- 对后续 PSCAD 外部原型，建议先采用保守且可解释的“低压状态持续时间超过当前判据允许时间则脱网”的简单计时器，并在实现前人工确认 `Vs=0.20` 边界。

## 11. 已确认事实、工程转译与待确认事项

### 已确认事实

- PDF 第 20 页包含图 2-3 和公式 (2-8)。
- 公式低电压内部段为 `0.20 < Vs < 0.90`。
- 图 2-3 的低压恢复边界从 `t0 + 0.625 s`、约 `0.20 pu` 线性升至 `t0 + 2 s`、约 `0.90 pu`。
- 公式第一行写明 `Vs <= 0.20` 时 `tVRT = 0`。

### 工程转译

- 对 `0.20 < Vs < 0.90`，可用线性函数计算允许持续时间。
- 对 `Vs > 0.90` 且低于高压分支，低电压计时器应退出或复归。
- 对 `Vs < 0.20`，公式支持立即脱网候选规则。

### 待确认事项

1. `Vs = 0.20 pu` 应按正文/图保留 `0.625 s`，还是按公式第一行立即脱网。
2. 后续外部 LVRT 原型中，计时器应采用瞬时电压实时限值、故障期间最小电压限值，还是其他论文实现方式。
3. `VIBR1_2` 的 POC 电压基准是否可直接等同论文 `Vs`，或是否需要额外缩放/滤波。

## 12. 下一步建议

由于 `Vs = 0.20 pu` 的边界存在正文/图/公式冲突，本轮建议暂不进入自动实现 LVRT 脱网逻辑。可以进入“故障持续时间边界扫描准备”的前置设计，但正式扫描前应先人工确认：

```text
Vs = 0.20 pu 时，外部 LVRT 原型应采用 0 s 立即脱网，还是 0.625 s 保持并网。
```

在该边界确认后，才能将低电压判据升级为 `ready_for_external_prototype`。
