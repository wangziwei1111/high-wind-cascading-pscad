This repository's own scripts, configuration, and documentation are MIT licensed.

External model redistribution policy:

- PNNL model files are not committed because the cloned repository did not contain an explicit LICENSE file during this audit.
- PSCAD official benchmark and WTG examples are not committed because PSCAD/MHI access and terms must be followed by the user.
- E-TRAN runtime/library files are not committed.
- User machine license files, compiler artifacts, and PSCAD generated binaries are not committed.

Allowed in Git: source names, URLs, commit hashes, file lists, SHA-256 hashes, manual acquisition instructions, and user-authored MATLAB/doc/config files.
