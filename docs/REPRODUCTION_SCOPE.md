# Reproduction Scope

This project is a **public-model-based independent reproduction framework**, not a claim of complete thesis replication.

In scope for Phase 1:

- Source audit and provenance.
- Parameter extraction with source labels.
- MATLAB protection logic scaffold and tests.
- Manual PSCAD assembly plan.
- PSCAD GUI validation checklist.

Out of scope for Phase 1:

- Automatic PSCAD schematic generation.
- Automatic `.pscx`, `.pslx`, `.psl`, or `.lib` creation or modification.
- Claiming PSCAD compile, initialization, or dynamic simulation results.
- Redistributing PSCAD official or license-unclear third-party models.
